Contract

Your class must subclass BaseStrategy. Required methods:

async def on_tick(self, tick: MarketTick, extra_data=None) -> TradeSignal | None
async def on_bar(self, bar: OHLCVBar, extra_data=None) -> TradeSignal | None

MarketTick fields

tick.symbolstrTicker symbol (e.g. "BTCUSDT", "AAPL")
tick.pricefloatLast trade price
tick.volumefloatTrade size / quantity
tick.timestampdatetimeUTC timestamp
tick.exchangeExchangeBINANCE or ALPACA

OHLCVBar fields

bar.symbolstrTicker symbol
bar.openfloatBar open price
bar.highfloatBar high price
bar.lowfloatBar low price
bar.closefloatBar close price
bar.volumefloatTotal volume in bar
bar.intervalstrBar interval (e.g. "1m", "5m", "1h")
bar.timestampdatetimeUTC timestamp of bar close

extra_data (custom pipeline)

Optional dict from data/custom_data.py. Keyed by symbol:

extra_data.get("AAPL", {}).get("put_call_ratio")

Returns None if no custom pipeline is configured. Edit data/custom_data.py to add your own data sources.

TradeSignal (return value)

TradeSignal(
    symbol="BTCUSDT",
    signal=Signal.BUY,    # BUY, SELL, or HOLD
    strength=0.8,         # 0.0 to 1.0
    strategy_id=self.strategy_id,
    metadata={"reason": "..."},  # optional dict
)

Allowed imports

math, statistics, collections, itertools, functools, datetime, decimal, numpy, pandas, shared.enums, shared.schemas, strategy.base
